# Gaiden Engine & Game
**A Lightweight 2D Game Engine and Adventure Game built from scratch in Java.**

## ⚡️ Project Overview
**Gaiden** is a custom 2D engine and demonstration game developed as a semester project at **CTU Prague (FEL)**. The core objective was to architect fundamental game systems—rendering, physics, and state management—entirely from scratch, avoiding heavy external frameworks like LibGDX or Unity.

The project demonstrates advanced Object-Oriented Design (OOD) and the application of architectural patterns to solve common real-time simulation challenges.
